# Gmail API Control (Python)

Repositorio base para controlar un buzón Gmail con Python:
- Leer mensajes
- Enviar correos
- Refresh automático del access token usando refresh token
- Uso seguro en producción (sin commitear secretos)

## Requisitos
- Python 3.10+
- Proyecto en Google Cloud con Gmail API habilitada
- OAuth Client tipo **Desktop App**

Instalar dependencias:

```bash
pip install -r requirements.txt
```
